Download Support

If downloads aren't working:

  • Try right-clicking the button and select "Save link as" or "Save as"
  • Check your Downloads folder — the file may have downloaded automatically
  • Look for a download icon in your browser's address bar (small down arrow)
  • Clear your browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
  • Try a different browser (Chrome, Safari, Firefox, Edge)

Opening the files:

  • Word files (.docx): Microsoft Word, Google Docs, Pages, LibreOffice
  • Excel files (.xlsx): Microsoft Excel, Google Sheets, LibreOffice Calc
  • HTML files: Any web browser (Chrome, Safari, Firefox, Edge)
