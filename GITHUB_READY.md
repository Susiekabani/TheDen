## 🔗 How Downloads Work

When Dragons click download buttons:

1. **File is in GitHub root directory**
2. **HTML link uses relative path** (e.g., `href="The_Den_Dragon_Scoresheet.xlsx"`)
3. **File downloads directly from GitHub**
4. **No external hosting needed**

This is why all files MUST be in the root of your repository.
